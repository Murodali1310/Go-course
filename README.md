## Go Courses / Practice Projects

Набор учебных проектов на Go: консольные утилиты, HTTP‑сервисы и реализации базовых примитивов синхронизации через каналы.

### Структура репозитория

- **gitfame** — CLI‑утилита для анализа вклада авторов в git‑репозиторий. Считает количество строк, уникальных коммитов и файлов по данным `git blame`/`git ls-tree`,
- поддерживает фильтрацию по расширениям, языкам и glob‑паттернам, сортировку по разным метрикам и вывод в форматах tabular/CSV/JSON/JSON‑lines.

- **digitalclock** — HTTP‑сервер, который по запросу `/?time=hh:mm:ss&k=N` генерирует PNG‑картинку с цифровыми часами.
 Поддерживает масштабирование изображения (`k`) и возврат текущего времени при пустом параметре `time`.

- **urlshortener** — сервис укороченных ссылок. Предоставляет API `POST /shorten` (принимает URL и возвращает ключ) и `GET /go/<KEY>` (302 редирект на исходный URL).
 Состояние хранится в памяти с защитой конкурентного доступа.

- **rwmutex** — собственная реализация `sync.RWMutex`, построенная на каналах (без использования пакета `sync`). Обеспечивает совместный доступ множества читателей и эксклюзивный доступ писателя к разделяемым данным.

- **waitgroup** — реализация аналога `sync.WaitGroup` через каналы. Позволяет ждать завершения группы горутин, соблюдая те же инварианты, что и стандартный примитив.
 
- **once** — реализация аналога `sync.Once` через каналы для ленивой инициализации, гарантирующая, что переданная функция будет выполнена ровно один раз при конкурентных вызовах.
